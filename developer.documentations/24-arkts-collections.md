# @arkts.collections (ArkTS Collections)

> **Source:** https://developer.huawei.com/consumer/en/doc/harmonyos-references/js-apis-arkts-collections

## Overview

The `@arkts.collections` module provides ArkTS-native collection types. These are the primary collection data structures for ArkTS development.
