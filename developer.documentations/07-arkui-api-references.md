## Overview

This is the main index page for ArkUI API references. It lists all the API categories available for ArkUI development.

- [JavaScript Components](https://developer.huawei.com/consumer/en/doc/harmonyos-references/arkui-js-comp) - JavaScript-compatible web-like component references
- [C API](https://developer.huawei.com/consumer/en/doc/harmonyos-references/arkui-c) - NDK C APIs for native UI development
- [Error Codes](https://developer.huawei.com/consumer/en/doc/harmonyos-references/arkui-arkts-errcode) - ArkUI error code references
- [es2abc Compiler Error Codes](https://developer.huawei.com/consumer/en/doc/harmonyos-references/errorcode-es2abc) - Compiler error codes
