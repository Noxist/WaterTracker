# Window Manager

> **Source:** https://developer.huawei.com/consumer/en/doc/harmonyos-guides/window-manager
> **Last updated:** 2026-01-14

## Overview

The Window Manager provides comprehensive window management capabilities for HarmonyOS applications, including window creation, configuration, rotation, multi-window support, and PiP (Picture-in-Picture).
